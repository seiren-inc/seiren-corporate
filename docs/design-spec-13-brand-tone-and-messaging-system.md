# 清蓮コーポレートサイト デザイン設計書 13

## ブランドトーン / メッセージング統一設計

# トーン定義

静か / 丁寧 / 実務的 / 断定しすぎない / 説明過多にしない

# 禁止ワード

必ず / 絶対 / 最安 / No.1 / 完全保証 / 無料（事実除く）/ 今すぐ

# 避ける表現

過度な感情訴求 / 不安を煽る / 競合比較 / 数字の誇張 / 「整理」「整える」多用

# 推奨語彙

ご相談 / サポート / ご案内 / 承ります / 対応いたします / ご提案いたします

# 文体統一

- です・ます調 / 文末を揃える / 1文は最大60文字以内

# 見出し設計

- H1: 14文字×2行以内 / 断定・抽象避ける
- H2: 6〜14文字 / 端的・具体的

# リード文: 40〜80文字 / 長文化禁止 / 感情過多禁止

# 事業紹介文: 詳細は専門サイトへ / 価格・手順は書かない / 立場と対応範囲を説明

# 電話導線: 「お電話でのご相談」推奨 / 「今すぐ」「お急ぎください」禁止

# 強みの見せ方: 実務対応 / 提携体制 / 来店型相談 / 数字訴求禁止

# 法務ページ: 簡潔 / 断定 / 感情表現なし
