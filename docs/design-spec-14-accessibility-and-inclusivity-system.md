# 清蓮コーポレートサイト デザイン設計書 14

## アクセシビリティ / インクルーシブ設計

# 基本方針: WCAG 2.1 AA準拠目標

# ランドマーク: header / nav(aria-label) / main(id=main-content) / footer

# スキップリンク: body直下 / Tab時のみ表示 / 背景 brand-primary

# コントラスト: 本文 4.5:1以上 / 小テキスト 3:1以上

# フォーカス: :focus-visible / outline 2px solid brand-primary / offset有

# ボタン: 高さ44px以上

# フォント: 16px未満禁止（小テキスト14pxまで）

# prefers-reduced-motion: reveal/clip/progress無効化

# 画像alt: 装飾=alt="" / 情報=具体記述

# フォーム: label明示 / for紐付け / エラー明確

# キーボード: Tab順自然 / ナビ操作可能

# 禁止: 文字のみ状態変化 / 色のみ区別 / 小リンク / hoverのみ情報提示
