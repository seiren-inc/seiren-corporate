# 清蓮コーポレートサイト デザイン設計書 05

## 3-2. duration制限

- hover：200〜300ms
- reveal：600ms
- 最大700ms

---

# 4. prefers-reduced-motion

必須実装。

reduceの場合：

- reveal無効
- clip無効
- progress非表示
- hover最小化

JSとCSS両方で対応。

---

# 5. パフォーマンス制約

- 原則Vanilla JS
- 重いアニメーションライブラリ禁止
- Hero画像はLCP最適化
- 画像にwidth/height指定

---

# 6. ブランド配色運用

エフェクトで使用可能：

- brand-primary：ライン/hover/進行
- brand-accent：小ドット1箇所まで
- brand-support：補助線のみ

禁止：

- ピンク発光
- ネオン演出
- 多色グラデ

---

# 7. 禁止事項（最新サイトでも不採用）

- 3D回転
- 背景動画
- パララックス強
- カウントアップ数字
- 無限ループアニメ
- 文字の過度な分解演出

---

# 8. 成功基準

- 最新感がある
- 派手ではない
- iPhoneで滑らか
- 信頼業種として違和感がない
- 動きが記憶に残らない（体験は残る）

---

# 9. 本設計の役割

本設計はトップページのモダン演出の標準規格。
他ページへ横展開する際も本仕様を基準とする。
