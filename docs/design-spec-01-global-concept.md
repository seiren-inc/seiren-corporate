# 清蓮コーポレートサイト デザイン設計書 01

## 全体デザインコンセプト定義（Apple思想寄せ / ロゴ配色準拠）

### 1. 目的

清蓮コーポレートサイトを「静けさ × 信頼 × 実在性 × 上質な余白」で再設計する。  
営業サイト化は禁止。ブランド体験としての“品”を優先する。

---

### 2. ブランドポジション整理

**業種特性**

- 供養・終活領域（信頼業種）
- 実店舗来店あり（完全予約制）
- 電話問い合わせ中心
- サービス詳細は各専門サイトに委譲（コーポレートは会社と事業紹介）

**デザイン方向性**

- 医療・法律・大手企業サイトの“信頼構造”を踏襲
- Apple的な余白設計と簡潔な言葉
- 説明過多を避け、情報の静かな提示を優先

---

### 3. ブランドトーン

| 要素   | 指針                                     |
| ------ | ---------------------------------------- |
| 世界観 | 静かな水面／澄んだ空気                   |
| 感情   | 落ち着き・誠実・安心                     |
| 動き   | 控えめ、邪魔をしない                     |
| 言葉   | 簡潔、断定しすぎない                     |
| 色     | ニュートラル主役、ロゴ色はアクセント運用 |

---

### 4. デザイン原則（絶対ルール）

1. **余白は主役**：情報を詰めず“間”で格を上げる
2. **色は控えめ**：ロゴ色はアクセント、面積を増やさない
3. **動きは演出ではない**：気づくか気づかないかの補助
4. **説明しすぎない**：詳細は各専門サイトへ誘導
5. **実在性を担保**：写真・住所・電話・予約制の明示を重視

---

### 5. 配色設計（ロゴ準拠・トークン化）

**CSSトークン（必須）**

```css
:root {
  /* neutrals (Apple-like) */
  --bg: #ffffff;
  --surface: #f5f7f8;
  --text: #0f1720;
  --muted: #5f6b75;
  --line: rgba(15, 23, 32, 0.1);

  /* brand (from Seiren logo) */
  --brand-primary: #2f8c9c; /* Seiren Teal：主役（ロゴ文字） */
  --brand-primary-2: #246b77; /* hover/active */
  --brand-accent: #c9789b; /* Lotus Pink：極小（花） */
  --brand-support: #8fbf74; /* Soft Sage：極小（リング） */
}
```
