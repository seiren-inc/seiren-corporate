# 清蓮コーポレートサイト デザイン設計書 07

## タイポグラフィ / 視覚アイデンティティ統合設計

---

# 1. 目的

清蓮サイトの「品」はタイポグラフィで決まる。
・文字サイズの設計 / 行間 / ウェイト / 視線の流れ でブランドの格を作る。

---

# 2. フォント方針

- 日本語：Noto Sans JP（可読性重視）
- 英字：日本語フォントに統一（混在禁止）
- serifは使用しない

# 3. フォントウェイト設計

- 400（本文） / 500（補助） / 600（H2） / 700（H1のみ）
- これ以外は使用禁止

# 4. タイポスケール

- H1: clamp(32px, 4vw, 56px) / weight:700 / line-height:1.2 / 最大2行
- H2: 28〜32px / weight:600 / line-height:1.4
- H3: 20〜22px / weight:600
- 本文: 16〜17px / line-height:1.8 / color:--text
- 小テキスト: 13〜14px / color:--muted

# 5. 行間設計

- H1：1.2 / H2：1.4 / 本文：1.8 / 英字アイブロウ：1.6

# 6. 段落設計

- 段落間：24px
- 長文は最大720〜800px幅
- 中央寄せはHeroのみ許可

# 7. 強調ルール

- 可：太字(600) / brand-primaryの色変更（リンク）
- 禁：蛍光色 / 下線多用 / 斜体 / 文字サイズバラバラ

# 8. 視線設計

H2 → リード → 本文 → CTA の順で自然に落ちる

# 9. 数字・電話表記

- font-weight: 600
- サイズは本文より大きめ（+2px程度）

# 10. 禁止事項

- 極端な字間 / 全大文字の多用 / 文字アニメの過剰演出
- フォントの混在 / セクションごとのフォント変更

# 11. 本設計の役割

全ページ共通規格。デザインの格は文字で決まる。
