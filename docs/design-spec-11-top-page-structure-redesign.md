# 清蓮コーポレートサイト デザイン設計書 11

## トップページ構造 再設計（最新版モダン構成）

---

# 1. 目的

情報の追加は行わず、構造と体験を引き上げることのみを目的とする。

# 2. 全体構成（順番固定）

1. Hero
2. 信頼即提示ブロック（電話・予約制）
3. 清蓮について（2カラム）
4. 事業領域（カード）
5. 清蓮の強み
6. 提携・取引先
7. 会社情報ミニマル表示
8. 最終CTA
9. フッター

# 3. Hero

- H1: 最大14文字×2行 / リード: 40文字以内 / CTA: 1〜6語
- 動き: H1→リード→CTA を段差フェード / 600ms以内 / blur+translateY

# 4. 信頼即提示ブロック

- 完全予約制 / 営業時間（毎日9:00〜18:00）/ 電話番号（クリック可）
- 装飾禁止 / 中央寄せ不可 / 静かな帯として表示

# 5. 清蓮について（About）

- 左：テキスト / 右：実在写真 / gap: 48px
- 長文禁止 / 2〜3段落 / 余白多め

# 6. 事業領域

- PC: 3列 / SP: 1列
- タイトル + 短い説明 + 詳細リンク
- アイコン禁止 / 過剰装飾禁止

# 7. 強みセクション

- セクションヘッダー + 3要素横並び
- 実務対応力 / 提携ネットワーク / 来店型相談
- 数字訴求禁止

# 8. 提携・取引先

- ロゴ横並び / グレースケール / hoverで通常色
- ロゴは小さく控えめ

# 9. 会社情報ミニマル表示

- 社名 / 所在地 / 電話 / 代表者名
- 罫線で区切る / ボックス化しない

# 10. 最終CTA

- 背景: --surface
- 文言: 「ご相談をご希望の方へ」
- Primary: お問い合わせ / Secondary: 電話する

# 11. SP最適化

- Heroは縦中央寄せ / clampで文字サイズ制御 / 余白80px基準 / ボタン100%幅

# 12. 禁止事項

カウントアップ / 背景動画 / 赤色ボタン / 無限アニメ / 長文ヒーロー
