# 清蓮コーポレートサイト デザイン設計書 08

## 信頼設計 / コンバージョン導線構造設計

---

# 1. 目的

売り込まずに自然に行動へ導く構造を定義する。
コーポレートサイトは「営業サイト」ではない。

# 2. コンバージョン定義

- 主CV：電話発信（telリンク） / お問い合わせページ遷移
- 副CV：事業紹介ページ閲覧 / 会社情報閲覧

# 3. 信頼業種におけるCV設計原則

1. 煽らない / 2. 緊急性を作らない / 3. 数字訴求しない
2. 限定訴求しない / 5. 派手なボタンを作らない

# 4. 電話導線設計

- 表示場所：トップ上部（信頼即提示ブロック）/ フッター / お問い合わせページ
- 視認性を確保（本文より大きめ）
- 赤ボタン禁止 / brand-primaryのみ

# 5. 来店導線設計

来店は必ず：

- 完全予約制明記
- 営業時間明記（毎日9:00〜18:00）
- 所在地表示
- 地図リンク

"準備された来店"に見せる。

# 6. CTA設計

- Primary：「事業紹介を見る」/ 背景: brand-primary
- Secondary：「お問い合わせ」/ アウトライン

# 7. フッターCV設計

フッター直前に「お問い合わせ」「電話する」の2導線のみ。

# 8. 信頼補強要素（必須）

- 代表者名 / 所在地 / 電話番号 / 提携カテゴリ表示

# 9. 禁止事項

- 「今すぐ」「無料」「最安」「No.1」/ 赤色ボタン / 点滅

# 10. 成功基準

- 押し売り感がない / 迷わない / 不安がない / 電話が自然に押せる
