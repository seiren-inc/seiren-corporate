# NAP Master Definition v1

## 概要
株式会社清蓮の SEO・GEO・MEO 施策において、会社名、住所、電話番号、URL、営業時間、サービス名称の正規表記を統一するためのマスター定義書。
サイト、Googleビジネスプロフィール、構造化データ、FAQ、口コミ導線、投稿文面、AI引用面で同一表記を維持することを最優先とする。

## 正規表記ルール

### 会社名の表記ルール
- 全チャネルの正式社名は `株式会社清蓮` に統一する。
- タイトルタグ、見出し、会社情報、構造化データ、GBP の事業者名、FAQ本文、投稿文面、レビュー依頼文面で同一表記を使用する。
- 外部サービス側で文字数や仕様制約がない限り、`株式会社 清蓮` のような空白入り表記は使用しない。

### 住所の表記ルール
- 正式住所は `神奈川県横浜市戸塚区戸塚町4170 高橋ビル1F` を使用する。
- 郵便番号付き表記が必要な媒体では `〒244-0003 神奈川県横浜市戸塚区戸塚町4170 高橋ビル1F` を使用する。
- 番地と建物名の間は半角スペース 1 つに統一する。
- 都道府県から建物名まで省略せずに記載する。ローカルSEO上、`横浜市戸塚区` のみの省略表記は基本禁止とする。
- Googleビジネスプロフィール、構造化データ、サイトフッター、会社概要ページ、問い合わせ導線で完全一致を優先する。

### 電話番号の使用ルール
- 正式電話番号は `045-881-9952` を使用する。
- `0458819952`、`(045)881-9952` などの別表記は使用しない。
- サイト本文、フッター、構造化データ、GBP、FAQ、投稿文面で同一表記を用いる。
- `tel:` リンク値は実装都合で数字のみでもよいが、表示テキストは必ず `045-881-9952` とする。

### 正式WebサイトURLと URL 表記ルール
- コーポレートサイトの代表URLは `https://seiren-inc.co.jp/` を使用する。
- 散骨クルーズ事業サイトを指す場合のみ `https://seiren.ne.jp/` を使用する。
- コーポレートサイトの構造化データ、canonical、BreadcrumbList、GBP の Web サイト欄では `https://seiren-inc.co.jp/` を基準とする。
- 別事業サイトURLを掲載する場合は、コーポレートサイトURLと混同しないよう用途を明示する。
- URL 使用時は以下を厳守する。
- `http` と `https` を混在させない。
- `www` 有無を混在させない。
- 末尾スラッシュの有無は正規URL定義に合わせて統一する。
- 構造化データの `url`、GBP の Web サイト欄、問い合わせ導線、各種参照ドキュメントで完全一致させる。

### 営業時間の記載ルール
- 営業時間の正式値は `月曜〜日曜 9:00〜17:00` とする。
- 営業時間を掲載する場合は、サイト、GBP、構造化データ、FAQ、投稿文面で同一値・同一表現を用いる。
- 人間向け表記は `月曜〜日曜 9:00〜17:00` を基本とする。
- 機械可読形式や構造化データでは `Mo-Su 09:00-17:00` を使用する。
- `年中無休`、`随時対応`、`24時間受付` など、実態確認なしの拡張表現は使用しない。

### FAQ・口コミ・投稿文面での記載ルール
- FAQ 回答文、レビュー依頼文、GBP 投稿文では、社名を `株式会社清蓮` に統一する。
- サービス名は本マスター記載の正式名称を優先し、1 文内で別名を混在させない。
- 住所・電話番号・URL を文面に含める場合は、本マスターの正規表記をそのまま使用する。
- 口コミ本文はユーザー生成コンテンツのため原文改変しない。ただし、返信文や依頼文テンプレートでは正規表記を使用する。
- AI 検索や引用対策のため、定義文・FAQ・会社紹介文では会社名と主要サービス名を省略せず記載する。

## チャネル別適用ルール

### Googleビジネスプロフィールとの整合ルール
- 事業者名は `株式会社清蓮` を使用する。
- 住所、電話番号、WebサイトURL は本マスターと完全一致させる。
- カテゴリや説明文で主要サービス名を記載する際も、本マスターの正式名称を優先する。
- GBP 側の営業時間は `月曜〜日曜 9:00〜17:00` と完全一致させる。
- GBP 側の営業時間を更新した場合は、同日中にサイト表記と構造化データ定義も照合する。

### 構造化データとの整合ルール
- `name` には `株式会社清蓮` を設定する。
- `address`、`telephone`、`url`、`openingHours` は本マスターの正規値のみを使用する。
- `sameAs` や外部参照URLを追加する際も、公式確認済み情報のみ採用する。
- 構造化データ内で略称や別名を新設しない。

### SEO・GEO・MEO 共通ルール
- SEO 観点: タイトル、説明文、会社概要、フッターで NAP とサービス名の表記を統一する。
- GEO 観点: AI に引用されやすい定義文、FAQ、紹介文で `株式会社清蓮` と正式サービス名を明示する。
- MEO 観点: GBP、口コミ返信、投稿、外部媒体で住所・電話番号・URL の完全一致を優先する。

### サービス名の表記ルール
- サービス一覧、FAQ、投稿、会社紹介文では正式名称を優先して使用する。
- 文脈上必要な場合を除き、装飾語をつけた派生名称を見出しレベルで乱立させない。
- サービスを並列表記する場合は、正式名称の粒度をそろえる。
- `お墓じまい` と `改葬` は別サービス名として扱い、必要に応じて `お墓じまい・改葬` と併記する。

## 今後の更新ルール
- NAP またはサービス名を変更する場合は、本ファイルをマスターとし、先に更新してから各チャネルへ反映する。
- GBP、サイト、構造化データのいずれかで変更が発生した場合は、24時間以内に相互照合する。
- 新しいサービス名、別名、訴求語を追加する場合は、正式名称か販促表現かを明確に分けて登録する。
- 未確定事項が確定した場合は、更新履歴に根拠と反映範囲を記載する。
